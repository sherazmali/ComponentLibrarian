# Component Librarian
Component Librarian is a lightweight, desktop application for organizing and managing reusable code components by design patterns. Built as an educational project for software engineering courses, it helps students and developers store, search, and retrieve code snippets with metadata.

Features

Add code components with metadata (name, pattern, language, description, code)

Search and filter components in real-time

View component details and code snippets

Delete components as needed

Data persists locally using SQLite database

Quick Start

For Windows Users (Easiest)

Download ComponentLibrarian.exe from the Releases page

Double-click the file to run

No installation required!

For All Platforms (Using Python)

Make sure you have Python 3.8+ installed

Download these three files:

database.py
controller.py
gui.py

Put all files in the same folder

Run: python gui.py

How to Use

Add Component: Click "Add Component" button and fill the form

Search: Type in the search box to filter components

Filter: Use dropdowns to filter by pattern or language

View: Click any component to see details

Delete: Select a component and click "Delete Component"

For Developers

Project Structure

database.py - Handles database operations (Model)

controller.py - Manages logic and validation (Controller)

gui.py - User interface (View)

Requirements
